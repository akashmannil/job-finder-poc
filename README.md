# JobMatch

A two-sided, proof-over-reach hiring platform POC. Built feature-by-feature with a documented
commit trail — start at **[docs/DEVLOG.md](docs/DEVLOG.md)** to follow the build.

> Full root documentation (overview, architecture, prompts) is finalized in the last commit.

## Quick start

```bash
npm install
cp .env.example .env.local   # add your ANTHROPIC_API_KEY (needed once AI features land)
npm run dev
```

Open http://localhost:3000.
